# How to development

> System requirement

- NodeJs version 19.\*
- Git

1. Clone project

```sh
git clone https://github.com/tapsu01/tips-ecommerce.git
```

2. Install dependencies

```sh
cd tips-ecommerce && yarn install
```

3. Run

```sh
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000)
